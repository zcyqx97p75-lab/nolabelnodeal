# No Label No Deal - Campaign Website

A mobile-first campaign website for contacting EU parliament members about origin labeling.

## Features

- 24 EU languages support
- Mobile-first responsive design
- Contact EU MEPs with email copy functionality
- Text suggestions for emails
- Anonymous counter
- No registration or login required

## Technology

- Vanilla JavaScript
- HTML5
- CSS3
- No frameworks or dependencies

## Local Development

1. Start a local server:
```bash
python3 -m http.server 8000
```

2. Open in browser:
```
http://localhost:8000
```

## Deployment

This project can be deployed to:
- Railway
- Netlify
- Vercel
- GitHub Pages
- Any static hosting service

## Files

- `index.html` - Main page
- `styles.css` - All styling
- `script.js` - All functionality
- `translations.js` - All translations for 24 EU languages
- `EU_Parlamentarier_aktuell_mit_Mail.csv` - MEP data
